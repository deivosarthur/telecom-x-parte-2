# Telecom X - Predicción de Cancelación de Clientes

Proyecto de Machine Learning desarrollado como parte del Challenge Telecom X.

El objetivo es construir modelos predictivos capaces de identificar clientes con mayor probabilidad de cancelar sus servicios.

## Modelos utilizados

- Regresión Logística
- Random Forest

## Métricas evaluadas

- Accuracy
- Precision
- Recall
- F1-score
- Matriz de confusión

## Conclusión

Se identificaron variables clave que influyen en la cancelación de clientes, como el tipo de contrato, el tiempo de permanencia y los cargos mensuales.
